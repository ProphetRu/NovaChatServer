Detailed Description

Manager for JSON Web Tokens (JWT) operations.

Provides functionality for generating, verifying, and managing JWT tokens for authentication and authorization. Supports both access and refresh tokens with configurable expiration times.

Note
For production environments, consider using Redis or a database for token blacklisting instead of in-memory storage.
