Detailed Description

Provides cryptographic hashing functions for password security.

Implements password hashing using OpenSSL's EVP interface with support for MD5 (legacy) and SHA-256 algorithms. Includes salt support for enhanced security.

Note
All methods are thread-safe (use local OpenSSL contexts)
Warning
MD5 is cryptographically broken - use only for legacy compatibility
