Detailed Description

Main server class that orchestrates all components of the chat application.

Manages server lifecycle including startup, graceful shutdown, and resource management. Coordinates SSL configuration, request routing, connection listening, and worker threads. Implements the main server loop and provides the top-level API for controlling the server.

Note
Thread-safe operations with atomic state management
Warning
Ensure proper shutdown sequence to avoid resource leaks
