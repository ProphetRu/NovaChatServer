Detailed Description

Provides validation and sanitization utilities for input data.

Contains static methods for validating user inputs (logins, passwords, UUIDs, etc.) and detecting security threats like SQL injection and XSS attacks. All methods are thread-safe and noexcept.

Note
This is a utility class with only static methods
