Detailed Description

Thread-safe singleton logger for application-wide logging.

Provides configurable logging to both console and files with multiple severity levels. Supports separate access logging for HTTP requests. Implements the singleton pattern to ensure a single logging instance.

Note
Thread-safe for concurrent logging from multiple threads
Warning
Must be initialized before use via initialize() method
