Detailed Description

Abstract base class for HTTP request handlers.

This interface defines the contract for all HTTP request handlers in the application. It provides common functionality for creating JSON responses, handling authentication, and validating request data while allowing derived classes to implement specific endpoint logic.

Note
All methods are thread-safe and exception-safe unless otherwise specified.
