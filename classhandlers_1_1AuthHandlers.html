Detailed Description

Handles authentication-related HTTP endpoints.

Implements user authentication, registration, token management, and account operations. This class provides endpoints for user registration, login, token refresh, logout, password changes, and account deletion.

Note
All methods are thread-safe and exception-safe unless otherwise specified.
